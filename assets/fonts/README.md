Poppins font files

Please add the Poppins TTF font files into this folder with the exact filenames below (these match the entries in `pubspec.yaml`):

- Poppins-Light.ttf (weight: 300)
- Poppins-Regular.ttf (weight: 400)
- Poppins-Medium.ttf (weight: 500)
- Poppins-SemiBold.ttf (weight: 600)
- Poppins-Bold.ttf (weight: 700)
- Poppins-ExtraBold.ttf (weight: 800)

You can download these from Google Fonts (https://fonts.google.com/specimen/Poppins) or include them from a licensed source. After adding the files, run:

flutter pub get

Then rebuild the app to ensure the bundled fonts are applied.
